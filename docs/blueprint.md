# **App Name**: Alpha Coders

## Core Features:

- Community Member List: Display a list of community members with their names and usernames as shown from the user list.
- Core Team Highlights: Highlight the core team members (Vortex-16, Dealer-09, Rouvik, PixelPioneer404) at the top of the community list or in a separate dedicated section.
- GitHub Project Showcase: Fetch and display community projects from the provided GitHub organization URL (https://github.com/orgs/Alpha4Coders).
- Project Filtering: Allow users to filter projects based on different technologies

## Style Guidelines:

- Primary color: Deep blue (#34495E) to evoke trust, stability, and community.
- Background color: Light gray (#F0F3F4), creating a clean and modern backdrop.
- Accent color: Bright cyan (#48C9B0), used for interactive elements and to highlight key information.
- Body and headline font: 'Inter', a sans-serif font known for its readability and modern aesthetic.
- Use minimalist, outline-style icons to represent project categories and member roles, providing a clean and intuitive user experience.
- Employ a card-based layout for project listings to promote easy scanning and clear visual separation.
- Incorporate subtle transitions and hover effects on project cards to provide visual feedback and enhance interactivity.
